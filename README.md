# Simple-Calculator

This is a simple calculator console app built in Python.

## Description

This program allows the user to enter two numbers and choose a math operation. The calculator can perform basic arithmetic operations such as addition, subtraction, multiplication, and division.

## Features

- Add two numbers
- Subtract two numbers
- Multiply two numbers
- Divide two numbers
- Continue calculating with a previous result
- Restart the calculator
- Quit the program

## How to Run

1. Make sure Python is installed on your computer.
2. Download or clone this repository.
3. Open the project folder.
4. Run the Python file:

```bash
python Calculator.py
